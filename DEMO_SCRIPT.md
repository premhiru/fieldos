## Setup

Use the Demo Workspace from the dashboard Pilot Setup panel. Reset it before each external demo so the aviation dataset starts from a known state.

## Walkthrough

1. Open Dashboard and show project health, open Action Items, recent evidence, and pilot setup progress.
2. Open Projects and select the taxiway lighting project.
3. Open Inbox and show the WhatsApp-style field updates with photo, voice, and document evidence.
4. Open Project Intelligence and show the generated weekly report and evidence gallery.
5. Open Search and ask: `What work needs review on taxiway lighting?`

## Close

Position FieldOS as the operating layer that turns messy field communication into searchable project intelligence, human-reviewed action, and defensible reporting.
